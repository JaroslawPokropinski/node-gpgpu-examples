# Node-gpgpu examples

This repository consists of usages of [node-gpgpu](https://github.com/JaroslawPokropinski/node-gpgpu) library.
To execute the examples in the colab change runtime type to GPU (`Runtime` -> `Change runtime type` -> `GPU`).
